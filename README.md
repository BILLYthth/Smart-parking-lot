# Smart-parking-lot
This project is one of my subject project that i built and simulated a parking lot applying microprocessor. 
In this microprocessor course, I learned about how to control, and apply its function on the project.
<img width="933" height="576" alt="image" src="https://github.com/user-attachments/assets/616a54fe-1845-454e-929a-18141b847a0e" />
it anounce the available spot to park the cars. 
Led screen will be placed at every intersection and let the driver know, where they can park their car, and how many spots left. 
<img width="1372" height="770" alt="image" src="https://github.com/user-attachments/assets/48e7b947-2435-44c2-8c25-8449d28a5799" />
It detect a car by using Ultra sonic sensor, detecting the distance if it's under 35 cm, the spot will be marked as parked. 
There are also a timer, to count how many minutes they can park.
